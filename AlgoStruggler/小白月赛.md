## A  AS的幸运单词

> $\hspace{15pt}$ $Algo$ 是什么？
>
> $\hspace{15pt}$ $Algo$ 就是 **"Algorithm"**！！！

### 题目描述

AS是一名业余的算法竞赛爱好者（蒟蒻），他认为单词 **Algo** 能给他带来好运。今天上英语课的时候，英语老师在黑板上写下了一个由大小写字母、**空格**、`','`、`'.'` 组成的字符串，他想知道能否从这个字符串里选出若干个字母（不必连续），重新排列后组成 **"Algo"**。

注意：**大小写敏感**，即 `'A'` 和 `'a'` 被视为不同的字符。

#### 输入描述：

第一行输入一个整数 $T$ $(1 \leq T \leq 10^4)$，表示测试数据组数。

接下来每组测试数据包含两行：
- 第一行一个整数 $N$ $(1 \leq N \leq 10^5)$，表示字符串的长度。
- 第二行一个长度为 $N$ 的字符串 $S$。

保证所有测试数据的 $N$ 之和不超过 $10^5$。

#### 输出描述：

对于每组测试数据，如果可以经过重新排列后组成 **"Algo"**，输出 **"Yes"**，否则输出 **"No"**。

#### 示例1

**输入**

```input
3
7
AlgoA B
12
Hello, Algo.
10
algorithm
```

**输出**

```output
Yes
Yes
No
```

#### 样例解释

- **第一组数据**：字符串为 `"AlgoA B"`，包含字符 `'A'`、`'l'`、`'g'`、`'o'`，可以组成 `"Algo"`，输出 `Yes`。
- **第二组数据**：字符串为 `"Hello, Algo."`，包含字符 `'A'`、`'l'`、`'g'`、`'o'`（注意有两个 `'l'`，`'A'` 和 `'a'` 不同），可以组成 `"Algo"`，输出 `Yes`。
- **第三组数据**：字符串为 `"algorithm"`，缺少大写字母 `'A'`，无法组成 `"Algo"`，输出 `No`。

## B AS的快递站

### 题目描述

AS在校园快递站兼职。快递站有 $N$ 个货架排成一排，编号 $1$ 到 $N$。第 $i$ 个货架上初始有 $A_i$ 个快递包裹。

现在有 $M$ 个同学陆续来取快递。第 $j$ 个同学会从编号为 $U_j$ 和 $V_j$ 的货架上各取走 $1$ 个快递（如果某个货架上已经没有快递了，则该货架不操作）。

小笨想知道，在所有同学取完快递后，快递站里**包裹数量最多的货架**编号是多少？

#### 输入描述：

第一行输入两个整数 $N$ 和 $M$ $(1 \leq N, M \leq 10^5)$，分别表示货架数量和同学数量。

第二行输入 $N$ 个整数 $A_1, A_2, \dots, A_N$ $(0 \leq A_i \leq 10^9)$，表示每个货架初始的快递数量。

接下来 $M$ 行，每行输入两个整数 $U_j$ 和 $V_j$ $(1 \leq L_j \leq R_j \leq N)$，表示第 $j$ 个同学取件的货架编号。

#### 输出描述：

输出一个整数，表示所有操作结束后，包裹数量最多的货架的编号。

#### 示例1

**输入**

`````input
5 3
3 2 5 1 4
1 3
2 5
1 4
`````

**输出**

``````output
5
``````

#### 样例解释

- 初始：`[3, 2, 5, 1, 4]`
- 第1个同学（取1-3）：`[2, 1, 4, 1, 4]`
- 第2个同学（取2-5）：`[2, 0, 3, 0, 3]`
- 第3个同学（取1-4）：`[1, 0, 2, 0, 3]`

所以包裹数量最多的货架编号是$5$ 。

## C AS的数字游戏

### 题目描述

小A和小S在玩一个数字游戏。黑板上写着一个正整数 $n$。

游戏规则如下：
1. 两人轮流操作，小A先手。
2. 每回合，当前玩家可以进行以下两种操作之一：
   - 将 $n$ 减去 $1$。
   - 如果当前 $n$ 是**偶数**，可以将 $n$ 除以 $2$。
3. 谁先将 $n$ 变为 $0$，谁就获胜。

两位玩家都绝顶聪明。给定初始数字 $n$，请你判断小A能否获胜。

#### 输入描述：

第一行输入一个整数 $T$ $(1 \leq T \leq 10^4)$，表示测试数据组数。

接下来 $T$ 行，每行一个整数 $n$ $(1 \leq n \leq 10^5)$，表示初始数字。

#### 输出描述：

对于每组测试数据，如果小A有必胜策略，输出 `"A"`；否则输出 `"S"`。

#### 示例1

**输入**
```input
6
1
2
3
4
5
6
```

**输出**

`````output
A
S
A
A
S
A
`````

#### 样例解释

- **$n=1$**：小A只能减 $1$，操作后数字变为 $0$，小A获胜。输出 `A`。
- **$n=2$**：小A有两种选择：
  - 减 $1$：数字变为 $1$，未到 $0$。轮到小B，小S减 $1$ 变为 $0$ 获胜，小A输。
  - 除以 $2$：数字变为 $1$，同样小B下一步减 $1$ 获胜，小A输。
    因此小A必败，输出 `S`。
- **$n=3$**：小A只能减 $1$，数字变为 $2$。轮到小S面对 $2$，由上可知面对 $2$ 的玩家必败，故小S必败，小A获胜。输出 `A`。
- **$n=4$**：小A可以选择：
  - 减 $1$ 变为 $3$：小S面对 $3$ 只能减 $1$ 变为 $2$；然后小A面对 $2$ 必败，从而小A输。
  - 除以 $2$ 变为 $2$：小S面对 $2$ 必败，因此小A获胜。
    小A会选择除以 $2$ 获胜，输出 `A`。
- **$n=5$**：小A只能减 $1$ 变为 $4$。小S面对 $4$ 有必胜策略，故小A必败，输出 `S`。
- **$n=6$**：小A可以选择：
  - 减 $1$ 变为 $5$：小S面对 $5$ 必败，小A胜。
  - 除以 $2$ 变为 $3$：小S面对 $3$ 必败，小A胜。
    两种操作都通向胜利，小A必胜，输出 `A`。

## D AS的波浪序列

### 题目描述

AS 定义一个序列为“波浪序列”，如果该序列满足：对于任意 $1 < i < n$，都有 $a_i > a_{i-1}$ 且 $a_i > a_{i+1}$（峰），或者 $a_i < a_{i-1}$ 且 $a_i < a_{i+1}$（谷）。换句话说，序列中的每个元素（除首尾外）都必须是局部极值。

现在，AS 手上有 $1$ 到 $n$ 这 $n$ 个整数各一个。他想用这些数构造一个波浪序列，并且希望序列中**峰**的个数恰好为 $k$。

请你帮他判断是否存在这样的排列。如果存在，输出**字典序最小**的那一个；如果不存在，输出 `-1`。

#### 输入描述

第一行输入一个整数 $T$ $(1 \leq T \leq 10^4)$，表示测试数据组数。

接下来 $T$ 行，每行两个整数 $n, k$ $(1 \leq n \leq 10^5, 0 \leq k \leq n)$。

保证所有测试数据的 $n$ 之和不超过 $10^5$。

#### 输出描述

对于每组测试数据：
- 如果存在满足条件的波浪序列，输出一行 $n$ 个整数表示字典序最小的排列。
- 如果不存在，输出 `-1`。

#### 示例

**输入**
```input
4
4 1
4 2
5 2
3 0
```

**输出**

`````output
1 3 2 4
-1
1 4 2 5 3
-1
`````



## E AS的校园网升级计划

### 题目描述

AS就读的大学的校园网由 $N$ 个教学楼和 $M$ 条光纤组成，构成一个连通的无向图。第 $i$ 条光纤连接教学楼 $u_i$ 和 $v_i$，维修费用为 $w_i$。

由于预算有限，学校决定只保留 $N-1$ 条光纤，使得网络仍然连通（即形成一棵生成树），并且希望**总维修费用最小**。

但是，由于教学楼属于是学校的核心，**编号为 $1$ 的教学楼必须恰好连接 $D$ 条光纤**。

请你计算满足条件的最小总维修费用。如果无法满足条件，输出 `-1`。

#### 输入描述

第一行输入三个整数 $N, M, D$ $(2 \leq N \leq 1000,\ 1 \leq M \leq 5000,\ 0 \leq D < N)$。

接下来 $M$ 行，每行三个整数 $u_i, v_i, w_i$ $(1 \leq u_i, v_i \leq N,\ 1 \leq w_i \leq 10^5)$，表示一条光纤及其维修费用。

保证图是连通的，没有重边和自环。

#### 输出描述

输出一个整数，表示满足条件的最小总维修费用。如果不存在满足条件的方案，输出 `-1`。

#### 示例1

**输入**

```input
4 5 2
1 2 1
1 3 2
1 4 3
2 3 4
3 4 5
```

**输出**

```output
7
```

#### 示例2

**输入**

```input
3 3 3
1 2 1
2 3 2
3 1 3
```

**输出**

```output
-1
```

#### 样例解释

* 对于示例2：1号楼最多只能连接2条边，无法满足度数为3的要求。

## F AS的校庆

### 题目描述

AS就读的学校为迎接校庆，在学校的主干道上方悬挂了一条长度为 $N$ 的LED灯带。灯带由 $N$ 个灯珠串联而成，每个灯珠可以发出红光（用 `R` 表示）或蓝光（用 `B` 表示）。

学院打算对灯带进行 $M$ 次调试操作，操作分为两种：

1. **翻转操作** `1 l r`：将区间 $[l, r]$ 内所有灯珠的颜色进行**互换**（`R` 变 `B`，`B` 变 `R`）。
2. **查询操作** `2 l r`：查询区间 $[l, r]$ 内，**颜色交替出现的极长子串**的最大长度。我们称一个子串是“交替子串”，如果其中任意相邻两个灯珠的颜色都不相同（例如 `RBR` 或 `BRB`）。注意：单个灯珠也算作长度为 $1$ 的交替子串。

小A需要你帮助编写一个程序，能够高效地处理这些操作，并回答每次查询的结果。

#### 输入描述

第一行输入一个整数 $T$ $(1 \leq T \leq 10^4)$，表示测试数据组数。

对于每组测试数据：

- 第一行输入两个整数 $N$ 和 $M$ $(1 \leq N, M \leq 10^5)$，表示灯珠数量和操作次数。
- 第二行输入一个长度为 $N$ 的字符串 $S$，仅由字符 `'R'` 和 `'B'` 组成，表示初始灯珠颜色。
- 接下来 $M$ 行，每行描述一个操作，格式为：
  - `1 l r`：表示翻转区间 $[l, r]$ 的颜色。
  - `2 l r`：表示查询区间 $[l, r]$ 内最长交替子串的长度。

保证 $1 \leq l \leq r \leq N$。

**数据规模约定**：所有测试数据的 $N$ 之和不超过 $2 \times 10^5$，$M$ 之和不超过 $2 \times 10^5$。

#### 输出描述：

对于每个类型为 `2` 的操作，输出一行一个整数，表示查询区间内最长交替子串的长度。

#### 示例

**输入**

```input
1
6 6
RRRRRR
2 1 6
1 2 4
2 1 6
1 3 5
2 1 5
2 1 6
```

**输出**

`````output
1
2
3
3
`````

#### 样例解释

- 初始全R，查询最长交替长度1（单珠）。
- 翻转2~4 → `R B B B R`，查询最长交替：位置1-2 `R B` 长度2。
- 翻转3~5 → `R B R R B`，查询最长交替：位置1-3 `R B R` 长度3。
